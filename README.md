# 🚚 GazeRoute — Логистический сервис для 4-метровых Газелей

**GazeRoute** — это нишевое Fullstack-приложение для поиска попутных грузов и свободных машин, ориентированное на владельцев малотоннажного транспорта (Газель 4м). 

Сайт адаптирован для водителей: темная тема для ночных рейсов, крупные элементы интерфейса для мобильных устройств и строгая фильтрация по габаритам кузова.

## 🚀 Основной функционал
- **Поиск грузов:** Фильтрация по городам отправления и прибытия.
- **База машин:** Список свободных водителей с указанием их текущего местоположения.
- **Личный кабинет водителя:** Возможность настроить параметры своей машины и переключать статус "Свободен/Занят".
- **Управление объявлениями:** CRUD (создание, чтение, обновление, удаление) грузов для авторизованных пользователей.
- **Безопасность:** Доступ к контактам и созданию объявлений только после регистрации.
- **Валидация:** Ограничение веса (до 2т) и объема (до 20м³), специфичное для данного класса машин.

## 🛠 Технологический стек
- **Backend:** PHP 8.2+, Laravel 11 (Breeze, Eloquent, MVC)
- **Frontend:** Tailwind CSS, Alpine.js, Blade
- **Database:** MySQL
- **Environment:** Docker (Laravel Sail), Vite
